Selectors Documentation

Selector:

A CSS selector selects the HTML elements you want to style.

Types of Selectors

  • Element Selector

    • The element selector selects HTML elements based on the element name (or tag) for example p, h1, div, span, etc.

  • Universal Selector

    • The Universal selector (*) in CSS is used to select all the elements in an HTML document. It also includes other elements which are inside under another element.

  • Grouping Selectors

    • This selector is used to style all comma-separated elements with the same style.

  • Id Selector

    • The id selector uses the id attribute of an HTML element to select a specific element. and id of the element is unique on a page to use the id selector.

  • Class Selector

    • The class selector selects HTML elements with a specific class attribute.

  • Attribute Selector

    • The attribute selector [attribute] is used to select the elements with a specified attribute or attribute value.
