Pseudo Class & Elements

Pseudo Classes

A pseudo-class is used to define a special state of an element.

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

selector:pseudo-class { property: value; }

The :first-child pseudo-class matches a specified element that is the first child of another element.

The :last-child pseudo-class matches a specified element that is the last child of another element.

:link
unvisited link
:visited
visited link
:hover
mouse hover
:active
selected link

Pseudo Elements

::after
Insert content after every element
::before
Insert content before every element
::first-line
Selects the first letter of every element
::first-line
Selects the first line of every element
::marker
Selects the markers of list items
::selection
Selects the portion of an element that is selected by a user
